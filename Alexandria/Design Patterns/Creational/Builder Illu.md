  Builder whenever a part of product should be built"
=> So the Director is not always needed, as the client
  can also do that perfectly ^38wrI39M

- The build() function inside each concrete Builder to return the product
either to the director, or directly to client
- IMHO, the Director class can be use to specify pre-built version that
is frequently used, or else everything can be done by client ^i3YJoz1d
